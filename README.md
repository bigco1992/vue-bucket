## **MABU (My Bucket List) Ver.1.0**
배포시 오류 수정중

2021.08.08 ~ 2021.08.15 (추가 기능이 생길 수 있습니다.)

사용자의 Bucket List를 조회, 작성, 수정, 삭제할 수 있는 Grid 형태의 미니 프로젝트입니다.<br>

## 사용한 기술

### Front-End

- Vue 2.6.14 + Vuex + Vue-router
- Webpack
- Axios
- Toast Pop-up Notification

### **Back-End**

- Node.js 14.17.3
    - JWT
    - Bcrypt
- MongoDB

### Deploy

- AWS-EC2
- Apache Tomcat
- Nginx
- Heroku
