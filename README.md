## **MABU (My Bucket List)**

2021.08.08 ~ 2021.08.15

사용자의 Bucket List를 조회, 작성, 수정, 삭제할 수 있는 Grid 형태의 미니 프로젝트입니다.

## 사용한 기술

### Front-End

- Vue 2.6.14 + Vuex + Vue-router
- Webpack
- Axios
- Toast Pop-up Notification

### **Back-End**

- Node.js 14.17.3
    - JWT
    - Bcrypt
- MongoDB

### Deploy

- AWS-EC2
- Apache Tomcat
- Nginx
- Heroku
